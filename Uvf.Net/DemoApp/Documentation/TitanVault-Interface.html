TitanVault Native Interface Documentation

TitanVault v1.0.0 - Native C-style exports for cross-language compatibility.
This library provides secure vault operations with UTF-8 byte array password handling for both Cryptomator V8 and UVF vault formats.

Constants and Error Codes

Return Codes:
TITAN_VAULT_SUCCESS = 0 TITAN_VAULT_ERROR_INVALID_PARAMETER = -1 TITAN_VAULT_ERROR_VAULT_NOT_FOUND = -2
TITAN_VAULT_ERROR_INVALID_PASSWORD = -3 TITAN_VAULT_ERROR_ACCESS_DENIED = -4 TITAN_VAULT_ERROR_VAULT_CORRUPTED = -5
TITAN_VAULT_ERROR_INSUFFICIENT_BUFFER = -6 TITAN_VAULT_ERROR_UNSUPPORTED_FORMAT = -7 TITAN_VAULT_ERROR_INTERNAL = -100
Vault Format Constants:
TITAN_VAULT_FORMAT_CRYPTOMATOR = 0 TITAN_VAULT_FORMAT_UVF = 1
KDF Method Constants:
TITAN_VAULT_KDF_PBKDF2 = 0 TITAN_VAULT_KDF_SCRYPT = 1

Utility Functions

IntPtr titan_vault_get_version()
Returns a pointer to a null-terminated UTF-8 string containing the TitanVault library version.
Use this to verify the library version. Remember to free the returned string with titan_vault_free_string().
IntPtr titan_vault_get_last_error()
Returns a pointer to a null-terminated UTF-8 string containing the last error message.
Call this after any function returns an error code to get detailed error information. Free with titan_vault_free_string().
int titan_vault_detect_vault_format(IntPtr vaultPathBytes, int vaultPathLength)
Detects the vault format at the specified path and returns the format constant.
Pass UTF-8 encoded vault path bytes. Returns TITAN_VAULT_FORMAT_CRYPTOMATOR or TITAN_VAULT_FORMAT_UVF, or negative error code.

Cryptomator Vault Operations

int titan_vault_create_cryptomator_vault(IntPtr vaultPathBytes, int vaultPathLength, IntPtr passwordBytes, int passwordLength)
Creates a new Cryptomator V8 vault at the specified path with the given password.
Pass UTF-8 encoded vault path and password bytes. Returns TITAN_VAULT_SUCCESS on success or negative error code.
IntPtr titan_vault_load_cryptomator_vault(IntPtr vaultPathBytes, int vaultPathLength, IntPtr passwordBytes, int passwordLength)
Loads an existing Cryptomator V8 vault and returns a vault handle for further operations.
Pass UTF-8 encoded vault path and password bytes. Returns vault handle on success or zero on error.
int titan_vault_change_cryptomator_password(IntPtr vaultPathBytes, int vaultPathLength, IntPtr oldPasswordBytes, int oldPasswordLength, IntPtr newPasswordBytes, int newPasswordLength)
Changes the password of an existing Cryptomator vault from old password to new password.
All parameters are UTF-8 encoded byte arrays. Returns TITAN_VAULT_SUCCESS on success or negative error code.

UVF Vault Operations

int titan_vault_create_uvf_vault(IntPtr vaultPathBytes, int vaultPathLength, IntPtr adminPasswordBytes, int adminPasswordLength, int encryptFilenames, int kdfMethod, int kdfIterations)
Creates a new UVF vault with admin password, filename encryption option, and configurable key derivation method.
Set encryptFilenames to 1 to enable filename encryption, 0 to disable. Use KDF constants for kdfMethod parameter.
IntPtr titan_vault_load_uvf_vault(IntPtr vaultPathBytes, int vaultPathLength, IntPtr userPasswordBytes, int userPasswordLength, IntPtr userIdBytes, int userIdLength)
Loads an existing UVF vault with user credentials and optional user ID hint for faster authentication.
UserID can be null/zero for automatic detection. Returns vault handle on success or zero on error.
int titan_vault_add_user(IntPtr vaultPathBytes, int vaultPathLength, IntPtr adminPasswordBytes, int adminPasswordLength, IntPtr newUserIdBytes, int newUserIdLength, IntPtr newUserPasswordBytes, int newUserPasswordLength)
Adds a new user to an existing UVF vault using admin credentials.
Admin password is required to add new users. All parameters are UTF-8 encoded byte arrays.
int titan_vault_remove_user(IntPtr vaultPathBytes, int vaultPathLength, IntPtr adminPasswordBytes, int adminPasswordLength, IntPtr userIdToRemoveBytes, int userIdToRemoveLength)
Removes an existing user from a UVF vault using admin credentials.
Admin password is required to remove users. Cannot remove the admin user itself.

File Operations

int titan_vault_read_file(IntPtr vaultHandle, IntPtr filePathBytes, int filePathLength, IntPtr buffer, IntPtr bufferSize)
Reads the entire contents of a file from the vault into the provided buffer.
BufferSize is both input (available space) and output (required space). Returns TITAN_VAULT_ERROR_INSUFFICIENT_BUFFER if buffer too small.
int titan_vault_write_file(IntPtr vaultHandle, IntPtr filePathBytes, int filePathLength, IntPtr buffer, int bufferSize)
Writes the entire contents of the buffer to a file in the vault, creating or overwriting as needed.
Creates parent directories automatically if they don't exist. File path uses forward slashes as separators.
int titan_vault_file_exists(IntPtr vaultHandle, IntPtr filePathBytes, int filePathLength)
Checks if a file exists in the vault at the specified path.
Returns 1 if file exists, 0 if it doesn't exist, or negative error code on failure.
int titan_vault_delete_file(IntPtr vaultHandle, IntPtr filePathBytes, int filePathLength)
Deletes a file from the vault at the specified path.
Returns TITAN_VAULT_SUCCESS on success or negative error code if file doesn't exist or deletion fails.

Directory Operations

int titan_vault_create_directory(IntPtr vaultHandle, IntPtr directoryPathBytes, int directoryPathLength)
Creates a new directory in the vault, including any necessary parent directories.
Creates the entire directory path if intermediate directories don't exist. Use forward slashes as path separators.
int titan_vault_directory_exists(IntPtr vaultHandle, IntPtr directoryPathBytes, int directoryPathLength)
Checks if a directory exists in the vault at the specified path.
Returns 1 if directory exists, 0 if it doesn't exist, or negative error code on failure.
int titan_vault_delete_directory(IntPtr vaultHandle, IntPtr directoryPathBytes, int directoryPathLength)
Deletes a directory from the vault, including all its contents recursively.
Use with caution as this permanently deletes the directory and all files/subdirectories within it.

Stream Operations

64-bit File Support:
Stream operations support 64-bit offsets and lengths, enabling handling of files larger than 4GB. Seeking, positioning, and length operations all use `long` (64-bit) values for maximum compatibility.
IntPtr titan_vault_open_read_stream(IntPtr vaultHandle, IntPtr filePathBytes, int filePathLength)
Opens a file in the vault for reading and returns a stream handle for sequential access.
Use for large files where you don't want to load everything into memory. Remember to close with titan_vault_close_stream().
IntPtr titan_vault_open_write_stream(IntPtr vaultHandle, IntPtr filePathBytes, int filePathLength)
Opens a file in the vault for writing and returns a stream handle for sequential writing.
Creates the file if it doesn't exist, overwrites if it does. Close with titan_vault_close_stream() to finalize.
int titan_vault_stream_read(IntPtr streamHandle, IntPtr buffer, int count)
Reads up to 'count' bytes from the stream into the buffer.
Returns the actual number of bytes read, which may be less than requested if end of file is reached.
int titan_vault_stream_write(IntPtr streamHandle, IntPtr buffer, int count)
Writes 'count' bytes from the buffer to the stream.
Returns the number of bytes written (should equal count) or negative error code on failure.
long titan_vault_stream_seek(IntPtr streamHandle, long offset, int origin)
Seeks to a specific position in the stream using 64-bit offset for large file support.
Origin: 0=Begin, 1=Current, 2=End. Returns new position or negative error code. Supports files >4GB.
long titan_vault_stream_get_position(IntPtr streamHandle)
Gets the current position in the stream as a 64-bit value.
Returns current position or negative error code. Essential for large file operations and random access.
long titan_vault_stream_get_length(IntPtr streamHandle)
Gets the total length of the stream as a 64-bit value.
Returns stream length or negative error code. Supports files larger than 4GB.
int titan_vault_stream_set_length(IntPtr streamHandle, long length)
Sets the length of a writable stream, truncating or extending as needed.
Only works on write streams. Use for pre-allocating space or truncating files. Supports 64-bit lengths.
int titan_vault_stream_flush(IntPtr streamHandle)
Flushes any pending writes to ensure data is committed to storage.
Important for write streams to ensure data persistence. Call before critical operations or closing.
int titan_vault_close_stream(IntPtr streamHandle)
Closes the stream and frees associated resources, finalizing any pending writes.
Always call this when finished with a stream to ensure data is properly written and resources are freed.

Advanced File Operations

int titan_vault_list_directory(IntPtr vaultHandle, IntPtr directoryPathBytes, int directoryPathLength, IntPtr entriesBuffer, IntPtr maxEntries)
Lists the contents of a directory in the vault, returning entry names in the provided buffer.
EntriesBuffer should be an array of IntPtr to receive string pointers. Free each string with titan_vault_free_string().
int titan_vault_get_file_info(IntPtr vaultHandle, IntPtr filePathBytes, int filePathLength, IntPtr fileSizePtr, IntPtr lastModifiedPtr)
Retrieves file information including size and last modified timestamp.
Pass null for parameters you don't need. LastModified is returned as Unix timestamp (seconds since epoch).
int titan_vault_move(IntPtr vaultHandle, IntPtr sourcePathBytes, int sourcePathLength, IntPtr destinationPathBytes, int destinationPathLength)
Moves or renames a file or directory within the vault from source path to destination path.
Can be used for both moving between directories and renaming within the same directory. Overwrites destination if it exists.
IntPtr titan_vault_read_all_text(IntPtr vaultHandle, IntPtr filePathBytes, int filePathLength)
Reads the entire contents of a text file as a UTF-8 encoded string.
Convenient for reading configuration files or small text files. Free the returned string with titan_vault_free_string().
int titan_vault_write_all_text(IntPtr vaultHandle, IntPtr filePathBytes, int filePathLength, IntPtr textBytes, int textLength)
Writes UTF-8 encoded text to a file in the vault, creating or overwriting as needed.
Convenient for writing configuration files or small text files. Text should be UTF-8 encoded bytes.
int titan_vault_append_all_text(IntPtr vaultHandle, IntPtr filePathBytes, int filePathLength, IntPtr textBytes, int textLength)
Appends UTF-8 encoded text to the end of an existing file in the vault.
Useful for log files or incremental data writing. Creates the file if it doesn't exist.

Advanced Vault Operations

int titan_vault_get_vault_users(IntPtr vaultPathBytes, int vaultPathLength, IntPtr adminPasswordBytes, int adminPasswordLength, IntPtr usersBuffer, IntPtr maxUsers)
Retrieves the list of users in a UVF vault using admin credentials.
UsersBuffer should be an array of IntPtr to receive user ID strings. Free each string with titan_vault_free_string().
int titan_vault_change_uvf_admin_password(IntPtr vaultPathBytes, int vaultPathLength, IntPtr oldPasswordBytes, int oldPasswordLength, IntPtr newPasswordBytes, int newPasswordLength)
Changes the admin password of a UVF vault from old password to new password.
Requires the current admin password for authentication. All other user passwords remain unchanged.
int titan_vault_change_uvf_user_password(IntPtr vaultPathBytes, int vaultPathLength, IntPtr adminPasswordBytes, int adminPasswordLength, IntPtr userIdBytes, int userIdLength, IntPtr newUserPasswordBytes, int newUserPasswordLength)
Changes a specific user's password in a UVF vault using admin credentials.
Admin can change any user's password without knowing the old password. Specify the user ID to change.
int titan_vault_rotate_keys(IntPtr vaultPathBytes, int vaultPathLength, IntPtr adminPasswordBytes, int adminPasswordLength, int vaultFormat)
Rotates (regenerates) the encryption keys for enhanced security, requiring admin credentials.
This is a security operation that generates new encryption keys while preserving access to existing data.
int titan_vault_backup_files(IntPtr vaultPathBytes, int vaultPathLength, IntPtr backupPathBytes, int backupPathLength, int overwriteExisting)
Creates a backup of vault configuration files to the specified backup directory.
Set overwriteExisting to 1 to overwrite existing backup files, 0 to fail if backup files already exist.
int titan_vault_close_vault(IntPtr vaultHandle)
Closes the vault handle and frees all associated resources.
Always call this when finished with a vault to ensure proper cleanup and resource deallocation.

Memory Management

void titan_vault_free_string(IntPtr stringPtr)
Frees a string that was allocated by TitanVault functions.
Call this for every string returned by TitanVault functions to prevent memory leaks.
void titan_vault_secure_zero_memory(IntPtr buffer, int size)
Securely zeros out a memory buffer to prevent sensitive data from remaining in memory.
Use this to clear password buffers or other sensitive data after use for enhanced security.
Security Notes:
• All password parameters should be UTF-8 encoded byte arrays
• Use titan_vault_secure_zero_memory() to clear sensitive data after use
• Always free strings returned by TitanVault with titan_vault_free_string()
• Close vault handles with titan_vault_close_vault() when finished
Path Format:
• Use forward slashes (/) as path separators for cross-platform compatibility
• Paths are relative to the vault root directory
• UTF-8 encoding is required for all string parameters
